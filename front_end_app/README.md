![预览图](https://github.com/LLL4040/images/blob/master/booklist%E9%A2%84%E8%A7%88.png?raw=true)

## 功能介绍

* 点击表头的“序号”或者“价格”可按照对应的升序排列

* 填写下面的表单可以向表格中添加书籍

* 点击“删除”可以删除对应的书籍信息

* 中间右侧的总计会自动计算书单里书籍的总价

* 增删书籍时，书单中的序号也会随之变动
